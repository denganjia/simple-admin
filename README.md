# Simple Admin

## 简介

Simple Admin 是一款基于 Vue3，Vite2，Typescript，NaiveUI 的基础后台管理模板，重点在于基础，用户权限需要你自己根据公司业务处理，其他的我都帮你做好了😁

## 特性

- 根据路由自动生成`menu`侧边导航和面包屑
- 可自己调整主题
- 全量使用`typescript`
- 基于`axios`的请求函数封装

## 立即使用
- 克隆代码
```bash
git clone https://github.com/denganjia/simple-admin.git
```
- 安装依赖
```
yarn
```
- 运行
```
yarn dev
```
- 打包
```
yarn build
```
